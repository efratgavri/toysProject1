Users

Get
/users/myInfo
get user info by token
Get
/users/usersList
Get list of all users - by token admin.
POST
/users
Add a new user to the data base.
                            Body
                            
                                {
                                    "name":"Yair",
                                    "email":"yyy@gmail.com",
                                    "password":"123456"
                                }
POST
/users/login
User login, if the email and password are correct the server will respond with a token, 
                        Body
                        
                            {
                                "email":"yyy@gmail.com",
                                "password":"123456"
                            }
DELETE
/users/{idDel}
Delete a user from the data base. Admin token can delete every user , and user can delete only himself.
PUT
/users/{idEdit}
Update your user in the data base, Admin token can edit every user data , and user can edit only himself. 
                            Body
                            
                                {
                                    "name":"Yair",
                                    "email":"yyy@gmail.com",
                                    "password":"123456"
                                }

Toys

Get
/toys
Get a list of all the toys, will present 10 results per page.
Get
/toys/search?s=doll
search for a toy by searching follwing name or info after the "s=", it will present 10 results per page.
Get
/toys/prices?min=50&max=200
Get a list of the toys by min & max , it will present 10 results per page.
Get
/toys/category/:catName
Get a list of all the toys in toys category, it will present 10 results per page
POST
/toys
Add a new toy to the data base, You need a token for this query
                            Body
                            
                                {
                                    "name":"Fish Game Doll",
                                    "info":"Fish,Doll",
                                    "category":"Doll",
                                    "img_url":"Fish.jpg" ,
                                    "price":25
                                }
POST
/toys/{idEdit}
Update a toy in the data base,You need a token for this query.
                            Body
                            
                                {
                                    "name":"Fish Game Doll",
                                    "info":"Fish,Doll",
                                    "category":"Doll",
                                    "img_url":"Fish.jpg" ,
                                    "price":25
                                }
.
DELETE
/toys/{idDel}
Delete a toy from the data base, You need a token for this query.
